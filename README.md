# MindFlow — Where am I?

Mind maps for your AI conversations. So you never lose track again.


## What it does
- Reads your chat with major AI assistants
- Generates a mind map of your thinking path
- Shows a recap of where you branched off

## Install
https://chromewebstore.google.com/detail/mindflow-where-am-i-in-ai/bdkdahkdnpljjnbkkcgfgbghpjidjoii

## How it works
Click the extension → it reads the current chat → sends it through your API key → renders the mind map.

Your conversation stays between you and your model provider.

## Stack
TypeScript, Vite, Markmap, Manifest V3
