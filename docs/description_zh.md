## 详细描述

你有没有过这样的经历：和AI聊着一个重要的知识点，一个问题接一个问题，突然聊到了一个全新的概念，于是你追问了几句，然后又追问了几句……二十分钟后你停下来想：我刚才到底在学什么来着？

MindFlow 回答的就是每个AI学习者都会问的那个问题：**我现在聊到哪了？**


🧠 什么是 MindFlow？

MindFlow 是一款 Chrome 浏览器扩展，能将任何 AI 对话转化为结构化的思维导图和复盘笔记——让你时刻知道自己在哪、聊过什么、每个话题之间是怎么串起来的。

它以可调节侧边栏的形式，直接显示在你的 AI 聊天页面旁边。无需切换标签页，一边聊天一边看图。


🎯 核心功能

• 思维导图生成 — 将整段 AI 对话转化为可视化的树状结构，展示每一个话题、每一个分支、每一次岔开。

• 结构化复盘笔记 — 在思维导图之外，MindFlow 同时生成学习笔记，解释每个概念是什么、为什么会在对话中出现——非常适合复习回顾。

• 灵活的侧边栏 — 拖动侧边栏调整大小，适配你的屏幕和工作习惯。支持局部放大、字体缩放，方便阅读。聊天的同时保持侧边栏打开，随时参照。

• 双视图模式 — 可以只打开思维导图、只打开笔记，或者两者并排显示。随时自由切换。

• Markdown 导出 — 一键导出或复制思维导图、笔记，或两者同时导出为 Markdown 格式——直接粘贴到 Obsidian、Notion、Logseq 或任何支持 Markdown 的笔记工具中。

• 多平台支持 — 支持所有主流 AI 聊天平台，包括ChatGPT, Claude, DeepSeek, 豆包, Gemini, Grok。

• 自带 API Key — 连接你自己的 AI 服务商（Anthropic, DeepSeek, 豆包, Google Gemini, OpenAI, 豆包）来驱动分析。你的 API Key 仅存储在浏览器本地，绝不会发送到我们的服务器。


📖 使用方法

1. 打开任意支持的 AI 聊天平台（ChatGPT、Claude、Gemini 等）。
2. 点击浏览器工具栏中的 MindFlow 图标。
3. 侧边栏在对话旁边打开。
4. 输入你的API Key。
5. 点击「生成」按钮，即可根据对话内容生成思维导图和笔记。
6. 调整侧边栏大小、放大查看具体分支，或在思维导图和笔记视图间切换。
7. 学习结束后，导出为 Markdown，保存你的学习路径。


💡 适合谁用？

• 用 AI 自学的人 — 经常在长对话中跑偏、忘记主线的自学者。
• 知识工作者 — 通过 AI 调研复杂课题，需要随时理清思路的专业人士。
• 学生 — 用 AI 辅助理解难点，需要结构化笔记做复习的学生。
• 知识管理爱好者 — 习惯用 Obsidian、Notion 等工具做笔记，但懒得把和 AI 聊天中学到的东西再手动敲一遍。
• 所有人 — 只要你曾经问了AI一个简单的问题，结果二十分钟后发现自己完全不记得最初在学什么。



🔒 隐私与数据

MindFlow 尊重你的隐私。以下是你的数据处理方式：

- API Key 使用 Chrome 的 storage API 存储在浏览器本地，绝不会传输到 MindFlow 的服务器。
- 对话内容仅在你点击「生成」时读取，并直接发送到你选择的 AI 服务商（如 OpenAI、Google）进行分析——MindFlow 不运营任何中间服务器。
- MindFlow 不收集、不存储、不分享你的任何对话数据。
- 你可以随时通过卸载扩展来删除所有本地存储的数据。

使用的权限：

- storage（存储）— 在浏览器本地保存你的 API Key 和偏好设置。
- activeTab（活动标签）— 在你点击 MindFlow 图标时，读取当前 AI 聊天页面的对话内容。

可选的 Host 权限（仅在使用 AI 模式时请求）：

- api.anthropic.com、api.openai.com、api.deepseek.com、ark.cn-beijing.volces.com、generativelanguage.googleapis.com — 直接连接你选择的 AI 服务商进行思维导图生成。

Content Script 运行在：ChatGPT、Claude、DeepSeek、Gemini、Grok、豆包、Perplexity — 仅用于读取对话文本和显示侧边栏。
