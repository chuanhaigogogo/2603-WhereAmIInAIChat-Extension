# MindFlow Usage Instructions

## Platform-Specific Notes

### DeepSeek (chat.deepseek.com)

DeepSeek uses virtual scrolling — only messages currently visible on screen exist in the page DOM. Messages scrolled out of view are removed from the page.

**Before generating a mindmap**, scroll to the very top of your conversation, then slowly scroll all the way back down to the bottom. This allows the extension to capture all messages. If you skip this step, the mindmap will only include messages that were visible on screen at the time of generation.
