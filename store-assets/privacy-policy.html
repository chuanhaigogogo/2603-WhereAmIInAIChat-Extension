MindFlow Privacy Policy

Last updated: 2026-04-06

What MindFlow Does

MindFlow is a browser extension that generates mind maps from your AI conversations on supported platforms (ChatGPT, Claude, Gemini). It helps you visualize the structure of what you're learning.

Data Collection

We do not collect any data. MindFlow has no backend server, no analytics, no telemetry. The extension developer never receives any of your data.

Data Access

MindFlow reads conversation text from AI chat pages you visit (ChatGPT, Claude, Gemini) to extract knowledge points and generate mind maps. This processing happens entirely within your browser.

Data Storage

  • Preferences: Your settings (extraction mode, API provider) are stored in Chrome's local storage on your device.
  • API Key: If you choose to use AI-enhanced mode, your API key is stored in Chrome's local storage on your device only. It is never transmitted to anyone except the AI provider you selected (Anthropic or OpenAI) when making extraction requests.
  • Conversation data: Processed in memory during your session. Not persisted after you close the tab.

All stored data is removed when you uninstall the extension.

Third-Party Services (Optional)

If you enable AI-enhanced mode and provide your own API key, MindFlow sends Q&A pairs from your conversation to the AI provider you select:

  • Anthropic API (privacy policy)
  • OpenAI API (privacy policy)

This is entirely opt-in. In free mode, no data leaves your browser.

Permissions

  • storage: Save your preferences locally
  • activeTab: Toggle the sidebar on the current tab
  • API host permissions (optional): Only requested when you enable AI-enhanced mode. Allows the extension to call the AI provider's API.

Data Deletion

To delete all MindFlow data: uninstall the extension, or go to Chrome Settings > Extensions > MindFlow > Remove. All local data is immediately deleted.

Contact

Questions about this policy? Open an issue on the MindFlow GitHub repository.
